# AI 情侣互动产品 - 设计说明文档

## 项目概述
本项目是一个基于 Vue 3 + TypeScript + Tailwind CSS 构建的 AI 情侣互动产品，采用温馨浪漫的视觉风格，模拟主流 AI 聊天产品的交互体验。

## 技术栈
- **前端框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI 框架**: Tailwind CSS
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **语言**: TypeScript

## 页面功能规范

### 1. 官网首页 (HomePage.vue)
- **功能**: 产品核心功能介绍、Hero 区域、立即体验按钮（跳转至伴侣广场）
- **视觉风格**: 温馨浪漫的渐变背景、玻璃拟态设计
- **交互**: 平滑滚动、hover 效果

### 2. 用户登录/注册页 (LoginPage.vue)
- **功能**: 手机号验证码登录、第三方登录（微信/QQ）、新用户注册
- **视觉风格**: 简洁优雅的卡片设计、柔和的动画效果
- **交互**: 表单验证、状态反馈

### 3. AI 伴侣匹配创建页 (CreatePage.vue)
- **功能**: 选择伴侣性格、自定义头像、设置偏好
- **视觉风格**: 温暖的色调、卡片式布局
- **交互**: 实时预览、步骤引导

### 4. 主聊天互动页 (ChatPage.vue)
- **功能**: 消息发送/接收、打字动画、快捷回复、话题建议
- **视觉风格**: 现代化聊天界面、消息气泡区分
- **交互**: 实时消息动画、输入框联想提示、消息状态标识

### 5. AI 人设个性化设置页 (SettingsPage.vue)
- **功能**: 修改头像、性格、对话风格等设置
- **视觉风格**: 分栏式布局、卡片式配置
- **交互**: 实时预览

### 6. 聊天历史记录页 (HistoryPage.vue)
- **功能**: 对话列表、时间筛选、搜索
- **视觉风格**: 列表式布局、清晰的信息展示
- **交互**: 无限分页加载

### 7. 个人中心页 (ProfilePage.vue)
- **功能**: 用户资料、订阅权益、通用设置
- **视觉风格**: 简洁的菜单式布局
- **交互**: 菜单切换

## 视觉规范

### 色彩系统
- **主色调**: 粉色系 (#f43f5e）、紫色系 (#a78bfa)
- **辅助色**: 温暖的橙色系 (#fb923c)
- **背景色**: 深色主题 (#0f172a)
- **玻璃态**: rgba(255, 255, 255, 0.1)

### 动画效果
- 消息滑入动画
- 按钮发光效果
