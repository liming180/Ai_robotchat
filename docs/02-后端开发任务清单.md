## ✅ 代码质量标准

### Java 代码标准
1. **代码规范**：符合 Alibaba Java 开发手册
2. **命名规范**：类名大驼峰，方法名小驼峰，常量全大写下划线分隔
3. **注释规范**：类和公共方法添加 JavaDoc 注释
4. **异常处理**：合理使用自定义异常，统一异常处理

### Python 代码标准
1. **代码规范**：符合 PEP 8 规范
2. **类型注解**：函数参数和返回值添加类型提示
3. **文档字符串**：公共模块和函数添加 Docstring

### 性能与安全标准
1. **响应时间**：普通 API 响应 < 200ms，AI 生成 < 10s
2. **SQL 注入防护**：使用 MyBatis Plus，不拼接 SQL
3. **密码安全**：BCrypt 加密存储
4. **XSS 防护**：输出内容适当转义
